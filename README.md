Vue Palette Generator 🎨
Vue Palette Generator — это интуитивное веб-приложение для создания, управления и экспорта цветовых палитр. Идеальный инструмент для дизайнеров, разработчиков и всех, кто работает с цветом.

✨ Возможности
🎨 Генератор палитр — создавайте гармоничные цветовые схемы
🔍 Умный поиск — находите цвета по названию или HEX/RGB кодам
📁 Экспорт — сохраняйте палитры в форматах JSON, PNG, CSS переменных
💾 Библиотека — храните и организуйте ваши палитры
🎯 Редактирование — тонкая настройка каждого цвета
📱 Адаптивность — работает на всех устройствах
🌈 Предпросмотр — визуализация палитры в реальном времени

Установка
bash
# 1. Клонируйте репозиторий
git clone https://github.com/ваш-username/vue-palette-generator.git

# 2. Перейдите в папку проекта
cd vue-palette-generator

# 3. Установите зависимости
npm install

# 4. Запустите приложение в режиме разработки
npm run dev

🛠️ Технологии
Vue 3 — Composition API, <script setup>

Vue Router — Навигация между страницами

Pinia (опционально) — Управление состоянием

SCSS — Препроцессор CSS

Vite — Сборка и разработка

ESLint + Prettier — Линтинг и форматирование

🎮 Использование
1. Генерация новой палитры
Перейдите на страницу Generator

Выберите базовый цвет

Выберите тип цветовой гармонии (аналогичная, комплементарная и т.д.)

Настройте количество цветов

Сохраните палитру в библиотеку

2. Работа с библиотекой
Просматривайте все сохраненные палитры

Редактируйте существующие

Удаляйте ненужные

Экспортируйте в нужном формате

3. Экспорт палитр
Доступные форматы:

JSON — для импорта в другие приложения

PNG — изображение палитры

CSS — CSS переменные

SCSS — SCSS переменные и миксины
